# IP Assignment Template / CLA

> Template for contributor IP assignment. A legal advisor should review before activating for corporate contributors.

## Developer Certificate of Origin (DCO) — Default

All contributions to JSF projects are covered by the DCO. Contributors sign off with:

    Signed-off-by: Full Name <email@example.com>

This certifies compliance with https://developercertificate.org

No additional agreement is needed for individual contributors.

## Full CLA — When Required

A full CLA is required when:
- A sub-project onboards outside corporate contributors
- A company contributes code on behalf of employees

## IP Ownership

All contributions accepted under Apache-2.0 (or sub-project override license). JSF holds no special IP assignment from contributors beyond what Apache-2.0 provides unless a separate CLA is executed.
