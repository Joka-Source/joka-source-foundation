# Joka Connect

> Internal LinkedIn and alumni network for IIM Calcutta — the successor to Joka Jodi.

## Overview

Joka Connect is the alumni networking and community platform for IIM Calcutta. It is the evolution of "Joka Jodi" — a richer, more exhaustive internal network that serves professional networking, alumni dating, and community building. It will be integrated with AlumERP as the social and community layer.

**Privacy note:** Dating/matchmaking features involve PII. See [PII-POLICY.md](PII-POLICY.md). Sensitive user data (preferences, photos, etc.) never lives in this org-wrapper repo — it belongs in the Joka Connect app database with its own consent flow.
