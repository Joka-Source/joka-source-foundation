## Archival Process

When a cell is retired (per `governance/voting-rules.md`):
1. `projects/{slug}/` → `archive/{slug}-{retire-year}/`
2. A `RETROSPECTIVE.md` is filed in `archive/{slug}-{retire-year}/`
3. An entry is appended to this table
4. The slug is added to the burned list below and never reused
